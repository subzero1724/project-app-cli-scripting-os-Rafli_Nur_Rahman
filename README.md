# Project CLI Transaksi Sepatu

Program CLI sederhana untuk melakukan transaksi pembelian sepatu.  
Pengguna dapat memasukkan nama pembeli, memilih jenis sepatu, menentukan jumlah, dan mendapatkan total harga otomatis.

## Fitur
- Input nama pembeli
- Pilihan sepatu (Nike, Adidas, Puma)
- Perhitungan total harga
